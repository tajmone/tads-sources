Details

Noun conjunctions. These words and groups of words can be used to separate noun phrases from one another. Note that these do not need to be exclusive to noun phrases - these can occur as command conjunctions as well; this list is separated from commandOrNounConjunction in case there are conjunctions that can never be used as command conjunctions, since such conjunctions, which can appear here, would not appear in commandOrNounConjunctions.
